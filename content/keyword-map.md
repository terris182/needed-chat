# needed.chat — Keyword Map (support-intent long-tail)

Organic SEO target queries. Intent = someone quietly searching for words/comfort,
not a clinical query. Each maps to an article. Group = topic cluster. Status:
`live` (published), `queued` (next batches).

Rules: never target self-harm-method or crisis queries (route those users to 988,
never optimize for them). Stay E-E-A-T honest: peer support, not treatment.
